
# Spring Boot Simple App


## Pages Included

| Route       | Page         | 
|-------------|--------------|
| `/`         | Home         | 
| `/about`    | About        | 
| `/services` | Services     | 
| `/profile`  | Profile      | 
| `/contact`  | Contact      | 

---

## Tech Stack

- **Java 21**
- **Spring Boot**
- **Bootstrap 5**
- **Gradle**

---

## Project Structure

```

src/main/
├─ java/com/example/demo/
│   └─ controller/      
└─ resources/
├─ templates/        
│   ├─ index.html
│   ├─ about.html
│   ├─ services.html
│   ├─ profile.html
│   └─ contact.html
└─ static/          

```

---

## Running the Application

From IntelliJ or terminal:

```

./gradlew bootRun

```

Then open:



[http://localhost:8080](http://localhost:8080)
